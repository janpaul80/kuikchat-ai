# Build handoff

Implement this project as a real, functional app (routing, state, data) using the project's framework.

Follow the established visual direction — these are the source of truth:
- `.delos/DESIGN.md` — art direction & rationale.
- `.delos/tokens.css` — design tokens. Port them into the real app's theme (CSS variables / Tailwind / shadcn tokens): colours, fonts, radius, spacing.

Match the direction closely — colours, type, spacing, layout. If no direction exists yet, call `design_direction` before writing any UI.
